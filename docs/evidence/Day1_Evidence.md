## 遇到的问题

### 1. nvm 环境变量问题
- **问题描述**：nvm 安装成功后，新终端无法识别 nvm 命令
- **原因分析**：安装程序未正确配置环境变量，或终端未刷新环境变量
- **解决方法**：手动配置 NVM_HOME 和 PATH 环境变量，重启终端后生效

### 2. corepack 权限问题
- **问题描述**：使用 `corepack prepare pnpm@11.9.0 --activate` 时出现权限错误（EPERM）
- **原因分析**：nvm 安装目录权限限制
- **解决方法**：改用 `npm install -g pnpm@11.9.0` 全局安装 pnpm

### 3. 终端环境隔离问题
- **问题描述**：在 AI 沙盒终端中配置的环境变量不会自动应用到用户新打开的终端
- **原因分析**：沙盒终端与用户终端环境隔离
- **解决方法**：重启 VS Code 或手动配置系统环境变量

### 4. tsconfig.app.json 报错问题
- **问题描述**：项目依赖未安装时，TypeScript 无法找到 `@vue/tsconfig/tsconfig.dom.json`
- **原因分析**：缺少 `@vue/tsconfig` 依赖包
- **解决方法**：安装项目依赖后问题解决
